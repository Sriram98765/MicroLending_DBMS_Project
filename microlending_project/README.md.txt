## Overview
This project is a relational database system for a micro-lending firm. It handles customers, loan applications, amortization schedules, and repayments.

## Tech Stack
* **Database:** MySQL
* **Tool:** MySQL Workbench

## How to Run
1. Open MySQL Workbench.
2. Run `03_create_tables.sql` to build the schema.
3. Run `05_seed_data.sql` to load sample data.
4. Run `06_crud_and_queries.sql` to see reports and KPIs.
